REST

Rest needs a generic functions for CRUD operations so we keep these functions inside api-sdk/rest/api_helpers/api_base_helper.dart file under ApiBaseHelper class and the exceptions for these api goes under api-sdk/rest/api_helpers/api_exception.dart file. api-sdk/rest/rest_api_handler_data.dart contains all the generic functions which app needs to access.

class RestApiHandlerData {
  static ApiBaseHelper _apiBaseHelper = ApiBaseHelper();
  static getData(String path) async {
    final response = await _apiBaseHelper.get('$path');
    return response;
  }

  static postData(String path, dynamic body) async {
    final response = await _apiBaseHelper.post('$path', body);
    return response;
  }
}

for post we need to send the path which will be under api-sdk/api_constants.dart and a body, for get we need to send the path only.

Now api-sdk/main.dart for api sdk will contain all the functions which app needs to access for example if app wants to access data for user it will call this function inside ApiSdk class.

static getUserData(int id) async {
    final response =
        await RestApiHandlerData.getData('${apiConstants["auth"]}/users/$id');
    return response;
  }

To call this function from anywhere we just have to call:

ApiSdk.getUserData(id);
